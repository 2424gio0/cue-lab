# CUE LAB — 단서와 기억 유지 실험

객관식·단답형·서술형을 한 번에 풀고, 즉시 성과와 지연 기억을 비교하는 교육용 웹앱입니다.

## 온라인 실행

https://2424gio0.github.io/cue-lab/

`main` 브랜치에 변경 사항이 올라오면 GitHub Actions가 정적 사이트를 빌드하여 GitHub Pages에 자동 배포합니다.

## 다른 PC에서 실행하기

### 준비물

- Node.js 22 이상: https://nodejs.org/

### 실행 순서

1. ZIP 파일의 압축을 풉니다.
2. 압축을 푼 폴더에서 터미널 또는 PowerShell을 엽니다.
3. 아래 명령을 차례로 실행합니다.

```bash
npm install
npm run dev
```

4. 터미널에 표시되는 `http://localhost:3000` 주소를 브라우저에서 엽니다.

같은 와이파이의 다른 기기에서도 접속시키려면 운영체제 방화벽과 네트워크 설정이 추가로 필요할 수 있습니다. 가장 간편한 공유 방법은 배포된 웹 주소를 사용하는 것입니다.

## 사용자가 조정할 수 있는 시간

- 지문 읽기 시간(초)
- 객관식·단답형·서술형 통합 즉시 검사 시간(초)
- 지연 검사까지 기다리는 시간(분)
- 지연 검사 풀이 시간(초)

## 기록

결과는 서버가 아니라 해당 브라우저의 로컬 저장소에 저장됩니다. 첫 화면 또는 최종 결과 화면에서 CSV 파일로 내보낼 수 있습니다.
